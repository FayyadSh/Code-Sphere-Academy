<h1 style="color: #01d2cf; margin-bottom: 30px;"> CodeSphere Academy 🚀 </h1>

A Next.js-based application where users can browse, search, and purchase courses. Built with modern technologies like **<span style="color: #01d2cf;">TypeScript</span>**, **<span style="color: #01d2cf;">Tailwind CSS</span>**, **<span style="color: #01d2cf;">Firebase</span>**, **<span style="color: #01d2cf;">Nodemailer</span>**, **<span style="color: #01d2cf;">Stripe</span>**, **<span style="color: #01d2cf;">Swiper Slider</span>**, and **<span style="color: #01d2cf;">Context API</span>**.

---

## Features ✨

1. **Browse Courses**:
   - Display a list of courses with **<span style="color: #01d2cf;">Swiper Slider</span>** for a smooth carousel experience. 🎡
   - Course details include title, description, duration, and price. 📚

2. **Search Courses**:
   - Implement a **<span style="color: #01d2cf;">search bar</span>** to filter courses by title, category, or instructor. 🔍

3. **Add Courses to Cart**:
   - Users can add courses to their **<span style="color: #01d2cf;">cart</span>**. 🛒
   - Cart state is managed using **<span style="color: #01d2cf;">Context API</span>**. 🧩

4. **Purchase Courses**:
   - Integrated with **<span style="color: #01d2cf;">Stripe</span>** for secure payments. 💳
   - After successful payment, send a confirmation email using **<span style="color: #01d2cf;">Nodemailer</span>**. 📧

5. **Authentication**:
   - User authentication (login/signup) using **<span style="color: #01d2cf;">Firebase Authentication</span>**. 🔐
   - Protected routes for authenticated users. 🛡️

6. **Database**:
   - Store course data, user data, and purchase history in **<span style="color: #01d2cf;">Firebase Firestore</span>**. 🗄️

7. **Responsive Design**:
   - Built with **<span style="color: #01d2cf;">Tailwind CSS</span>** for a responsive and modern UI. 🎨

8. **Dark Theme**:
   - Users can switch between **<span style="color: #01d2cf;">light</span>** and **<span style="color: #01d2cf;">dark</span>** themes. 🌞🌙
   - The selected theme is stored in **<span style="color: #01d2cf;">local storage</span>** for persistence across sessions. 💾

---

## Tech Stack 🛠️

- **Frontend**: **<span style="color: #01d2cf;">Next.js</span>**, **<span style="color: #01d2cf;">TypeScript</span>**, **<span style="color: #01d2cf;">Tailwind CSS</span>**
- **Backend**: **<span style="color: #01d2cf;">Firebase</span>** (Authentication, Firestore)
- **Payment Gateway**: **<span style="color: #01d2cf;">Stripe</span>**
- **Email Service**: **<span style="color: #01d2cf;">Nodemailer</span>**
- **State Management**: **<span style="color: #01d2cf;">Context API</span>**
- **Theme Management**: **<span style="color: #01d2cf;">Local Storage</span>**
- **API Handling**: **<span style="color: #01d2cf;">Next.js App Router</span>**
